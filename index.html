🔑 Configuração de Chave API

O FísicaGen utiliza a infraestrutura do OpenRouter para conectar com modelos de inteligência artificial avançados.

Como obter uma chave?
  1. Crie uma conta gratuita em openrouter.ai.
  2. Vá nas configurações e gere uma nova API Key.
  3. Cole a chave no campo abaixo para habilitar a geração de questões.

🔒 Segurança: Sua chave de API é gravada apenas no navegador local (localStorage). Ela nunca é enviada para nenhum servidor externo além do próprio endpoint oficial do OpenRouter.
